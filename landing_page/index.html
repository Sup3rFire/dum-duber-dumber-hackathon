Setup

Four steps. Zero synergy.

Install the temporary Firefox add-on, add your key, then point Decrapify at the web’s most needlessly verbose corners.

  1. 1

    Download & unzip

    Download the ZIP above and extract it somewhere you can keep it.

  2. 2

    Load it in Firefox

    Open about:debugging → This Firefox → Load Temporary Add-on, then select the extracted manifest.json.

  3. 3

    Add your API key

    In the extension popup, open Settings / API key, paste your OpenAI API key, and save it locally in Firefox.

  4. 4

    Decrapify a page

    Open the toolbar popup and switch the current site to Decrapify. It stays off everywhere else until you enable it.

Temporary means temporary: Firefox removes this type of add-on when it restarts, so reload the same manifest.json from about:debugging next time. Your API key is stored locally; page text goes to OpenAI only after you switch a site on.
